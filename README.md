# InitiativeAppRN

### Gitin käyttö
Aina ennen commitien puskemista masteriin, tee 

`$ git fetch`

Jos masterissa ei ole edeltäviä muutoksia, voit vapaasti pushata tuotoksesi.

Jos taas masterissa on muita muutoksia, tee seuraavat toimenpiteet:

`$ git stash`

`$ git pull --rebase`

`$ git push`

`$ git stash pop`
